About Media Queries

The media queries are a special syntax for CSS that allows us to define some styles that will only be applied in the case that defined conditions are met. We can assimilate them to optional lines of code, which will only be displayed for some users or devices. Min-width means greater than or equal to the width specified in that media query. Max-width means less than or equal to the width specified in that media query. Relative Units

Relative length units are relative to something else, perhaps the size of the parent element's font, or the size of the viewport. The benefit of using relative units is that with some careful planning you can make it so the size of text or other elements scale relative to everything else on the page. Em is relative to the font-size of the element (2em means 2 times the size of the current font). Vw is relative to 1% of the width of the viewport. Vh is relative to 1% of the height of the viewport.

Meta Viewport Tag

A viewport element gives the browser instructions on how to control the page's dimensions and scaling. The width=device-width part sets the width of the page to follow the screen-width of the device (which will vary depending on the device). The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.
