Inheritance in CSS

In CSS, inheritance controls what happens when no value is specified for a property on an element. When applied to an element in a document, a property with the value 'inherit' will use the same value as the parent element has for that property. The background colour of the div element is white, because the background-color property is set to white.

Specificity in CSS

Specificity is usually the reason why your CSS-rules don't apply to some elements, although you think they should. Every selector has its place in the specificity hierarchy. If two selectors apply to the same element, the one with higher specificity wins. A compound selector can mix the previous 3 types using more than one class, ID and/or tag. CSS allows you to group multiple selectors that share the same declaration. This optimization technique allows you to apply the same style to multiple elements to save space.

ID and Class Attributes

The only difference between them is that “id” is unique in a page and can only apply to at most one element, while “class” selector can apply to multiple elements.
